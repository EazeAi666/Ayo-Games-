# Ayo Games Suite 🇳🇬

A premium collection of classic board and card games, deeply rooted in Nigerian culture and refined for a modern digital experience.

Created by **Adesina EazeAi**.

## 🎮 Included Games

### 1. Ayo (Ncho)
The traditional Yoruba board game of strategy and calculation.
- **Rules:** Authentic sowing and capturing mechanics.
- **Modes:** Play against friends locally or challenge the AI.

### 2. Whot! (Naija Edition)
Nigeria's favorite card game with all the classic "special" rules.
- **Special Cards:** 
  - **1 (Hold On):** Play again.
  - **2 (Pick Two):** Next player draws 2.
  - **5 (Pick Three):** Next player draws 3.
  - **8 (Suspension):** Skip next player.
  - **14 (General Market):** Everyone draws 1.
  - **20 (Whot):** Request a suit.
- **Defense:** Stack penalties to pass them to the next player!

### 3. Ludo
The ultimate family race game.
- **Pro Rules:** Requires a 6 to start, independent dice movement, and safe zones (stars).
- **Extra Turns:** Roll a 6 to go again.

### 4. Chess
A professional-grade Chess implementation for the strategic mind.
- **Features:** Full move validation, castling, en passant, and pawn promotion.
- **AI:** Integrated AI with multiple difficulty levels.

## ✨ Features
- **Naija Based:** Authentic rules and cultural flavor.
- **Premium UI:** Sleek, dark-themed interface with smooth animations (Framer Motion).
- **Local Multiplayer:** Play with friends on the same device.
- **AI Opponents:** Challenging AI powered by advanced logic.
- **Integrated Rules:** "How to Play" guides available for every game.

## 🚀 Tech Stack
- **Frontend:** React + TypeScript
- **Styling:** Tailwind CSS + Lucide Icons
- **Animations:** Framer Motion
- **State Management:** Custom Game Service with Local Persistence
